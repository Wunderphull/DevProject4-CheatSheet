Tags for HTML Tables:

Table Tags

Tag Name Description
<table> Table The <table> element is used to create a table. The contents are written out row by row.
<thead> Table Head The heading element should sit inside the <thead> element.
<tbody> Table Body The body should sit inside the <tbody> element.
<tr> Table Row The <tr> element indicates the start of a row. It is followed by <td> elements. At the end of the row a closing </tr> element is used.
<td> Table Data Each cell of a table is represented using a <td> element. At the close of each cell a </td> element is used.
<tfoot> Table Foot The footer belongs inside the <tfoot> element.

Attribute Tags

Attribute Name Span
<colspan> Column Span The <colspan> attribute defines how many columns a <td> element should span.
<rowspan> Row Span The <rowspan> attribute defines how many rows a <td> element should span.
